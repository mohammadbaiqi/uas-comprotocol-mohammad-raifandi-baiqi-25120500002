## Deskripsi Project

Mini project ini bertujuan untuk mengimplementasikan REST API sederhana sebagai media komunikasi antara client dan server menggunakan protokol HTTP. Sistem yang dibuat berfungsi untuk mengelola data produk melalui operasi CRUD (Create, Read, Update, Delete).

REST API dikembangkan menggunakan Express.js dengan penyimpanan data sementara menggunakan file JSON. Pengujian dilakukan menggunakan Postman untuk memastikan seluruh endpoint berjalan sesuai spesifikasi.

---

## Tujuan

- Menerapkan konsep REST API.
- Mengimplementasikan operasi CRUD.
- Memahami penggunaan metode HTTP (GET, POST, PATCH, DELETE).
- Melakukan pengujian endpoint menggunakan Postman.
- Menerapkan struktur aplikasi backend yang modular.
